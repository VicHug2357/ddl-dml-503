Reemplazar Filas

La operación REPLACE en SQL reemplaza una fila existente en una tabla por una nueva fila. Es útil cuando necesitas actualizar datos pero no puedes saber de antemano si la fila existe.

REPLACE INTO tabla (columna1, columna2) VALUES (valor1, valor2);

Actualizar Filas

La operación UPDATE en SQL actualiza los datos de una fila existente en una tabla.

UPDATE tabla SET columna1 = nuevo_valor WHERE condición;

Ejemplo

Supongamos que tenemos una tabla llamada 'usuarios' y queremos actualizar la edad de un usuario con un nombre específico:

UPDATE usuarios SET edad = 30 WHERE nombre = 'Juan';
